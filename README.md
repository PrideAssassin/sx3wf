# CrackWifi
## 简介：
* wifi弱口令爆破
* 版本：v3.0

## 使用说明
* 1 扫描附近wifi名称
* 2 选择字典
* 3 选择需要破解的wifi信息，确认选中信息后，点击开始破解，等待破解成功即可。
